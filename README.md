# EDUPEAK: The Resilient Learning Management Ecosystem

![EDUPEAK Student Dashboard](https://picsum.photos/seed/readme-dashboard/1200/600)

## Overview

EDUPEAK is a modern, feature-rich Learning Management System (LMS) designed to provide a resilient and engaging educational experience for both students and teachers in environments with varying internet connectivity. Built with a robust, offline-first architecture, it ensures that learning never stops.

The platform features distinct portals for students and teachers, each tailored to their unique needs, from personalized learning paths and gamified quizzes to powerful analytics and content curation tools.

---

## ‚ú® Key Features

### For Students
- **Personalized Dashboard**: A central hub to track overall progress, points, and skill completion.
- **Skill Quests**: Engage in interactive quizzes on various subjects to earn points and master new skills.
- **Learning Modules**: Access structured lessons with detailed notes and content.
- **Progress Tracker**: A detailed view of performance across different subjects and skill quests.
- **Financial Resilience Hub**: Connect skills to real-world opportunities with a scholarship matching tool and application readiness checklists.
- **Certificates & Badges**: Earn and view certificates for completed courses and collect badges for achievements.

### For Teachers
- **Class Analytics Dashboard**: Get data-driven insights into student performance, skill mastery, and class-wide progress.
- **Student Progress Overview**: Track individual student metrics and identify areas where they might be struggling.
- **Custom Content Tool**: Create and submit localized examples to enrich the curriculum and make it more relevant to students.
- **Professional Development Hub**: Access training modules to enhance teaching skills at your own pace.

## üöÄ Getting Started

To get the project up and running on your local machine, follow these simple steps.

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or later)
- [npm](https://www.npmjs.com/) (or another package manager like yarn or pnpm)

### Installation & Setup

1.  **Clone the repository** (if you're working from a Git repo):
    ```bash
    git clone <your-repository-url>
    cd <project-directory>
    ```

2.  **Install dependencies**:
    ```bash
    npm install
    ```

3.  **Run the development server**:
    ```bash
    npm run dev
    ```

4.  **Open the application**:
    Navigate to [http://localhost:9002](http://localhost:9002) in your web browser to see the application live.
